note bottom of fix_latest
The alternative way of publishing packages used in the QuickFIX/J, a jar for each FIX protocol version.
This allows an application to depend on artefacts for only the FIX version that it uses.
Message, Component and Field classes are copied to each of these artefacts as required.
Package names are the same as in quickfixj-messages-all,
Plant UML does not render this correctly so the package names are changed.

end note
